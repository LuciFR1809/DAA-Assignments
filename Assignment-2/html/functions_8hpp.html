recursively finds the best possible partitions of points, given M penalty at each index, P input point set, Err error for all possible segments of points

Parameters
[in]jinteger, partitions have to be found for elements lying before this index
[in]Pvector of points, given input
[in]Mvector of double, stores the penalty of every index
[in]costdouble, constant cost
[in]Errvector of vector of double, stores the error values corresponding to every ( i, j ) pairs
[out]resultvector of vector of points, gives the final optimal partitions for the set of points
Note
result gives a vector of partitions where each partition is a vector of points belonging to the same partition.
