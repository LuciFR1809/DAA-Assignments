Timing Analysis

PRE-PROCESSING :
  O(Nlog(N)):Initial sorting based on x coordinate
SEGMENTED-LEAST-SQUARE :
  O(N3):time when finding error of all possible combinations of segments, if a best fit line is made through each one of them
  O(N) :The recurrence relation(with use of memoization)

where N = Number Of Coordinates
therefore the final time complexity of the whole algorithm is equal to
O(N3)

Space Analysis

M[] :
  O(N):the array which keeps all the penalty values
err[] :
  O(N2):keeps the sum of squared error for best fit lines of all combinations of segments
final[] :
  O(N):vector of vectors which keeps the final set of segments

where N = Number Of Coordinates
therefore the final space complexity of the whole algorithm is equal to
O(N2)
