Algorithm Analysis

Problem Statement

Implement the Segmented Least Squares Algorithm & a visualizer to show input points & fitted lines

Algorithm Abstract

segmented least square algorithm attempts to find the best fit line/ lines for a set of points with continuous x coordinates, by finding the best possible segments in the set of points for which the error given by best fit line is minimum.

Recurrance

OPT(j)=min1≤i≤j(ei,j+C+OPT[i-1])

OPT(j) denote the minimum cost for points p1, p2, …, pj
ei,j denote the minimum squared error for points pi, pi+1, …, pj
C is the input penalty multiplier

Intution

The key point in defining the recurrence is that the last point is part of the last segment hence, if we can find best possible partition for all points before the last segment we have solved the problem.

The concept of penalty is used to find which segments give least error, where penalty is calculated based on error given by best fit line of current segment , a constant cost function and errors by all the previous segments.
