Algorithm Analysis

Problem Statement

Given a set of rectangles the algorithm aims to find the area - defined as "measure" and the perimeter defined as "contour" of the union of the set.

Algorithm Abstract

The Algorithm is based on Divide and conquer paradigm. The logic behind the algorithm is to divide x-interval on the basis of number of the left and right edges. Base case is when a partition contains only one edge from the set of edges. The agorithm defines a new data structure named stripes which consists of an x-interval, a y-interval and x_union - a real number in case of calculating measure, a ctree whose leaf nodes store coordinates of an interval.

Intution

The x_union of Stripe is merged for both measure and contour problems.x_union for merge gives the length of x-interval for a given stripe which is included inside some rectangle, simply a multiplication of x_union with length of y-interval gives us the measure

The x_union of Stripe for contour problem is a ctree whose leaf node stores coordinate and also whether it is the staring point of the interval of]r the end point. Traversal of this tree gives us the coordinates , after which we make intervals and check for those which make contour pieces. The contour pieces gives all the horizontal pieces of the contour and all that is left is to connect them with vertical line segments
